# Installation Theme

To install this theme, simply use the theme-install command:

```bash
theme-install https://github.com/HANCORE-linux/blackgold-theme.git
```
## Vesktop-Theme
For Vesktop users, you can install this theme by copying the theme file to your Vesktop themes directory:
```bash
cp system24-blackgold.theme.css ~/.config/vesktop/themes/system24-blackgold.theme.css
```

## Neovim-Theme

## VSCode-Theme
