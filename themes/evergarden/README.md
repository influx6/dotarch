# evergarden

[Omarchy](https://omarchy.org/) port of the [Evergarden](https://everviolet.xyz/) theme.

![demo](./.github/assets/demo.png)

### Installation

```bash
git clone https://github.com/celsobenedetti/evergarden.git ~/.config/omarchy/themes/evergarden

# or 
git clone https://github.com/celsobenedetti/evergarden.git ~/local/evergarden
ln -s ~/local/evergarden/ ~/.config/omarchy/themes/evergarden
```
